# 본문 (아래부터 WP에 붙여넣기)

"우리 카톡만 보면 썸인지 아닌지 알 수 있을까?"  
heydaystar **카톡 감정 분석기**는 그 질문에 **숫자와 리포트**로 답해 주는 무료 도구예요.

주소: **https://app.heydaystar.co.kr**

---

## 어떤 걸 알려 주나요?

| 항목 | 설명 |
|------|------|
| **호감도 점수** | 0~100점, 관계 온도 한눈에 |
| **관계 유형** | 썸·연애 / 친구 / 업무 등 |
| **밀당·주도권** | 누가 더 대화를 이끄는지 |
| **핵심 카톡 순간** | 관계를 바꾼 대화 인용 + 해석 |
| **실전 팁** | 지금 상황에 맞는 다음 행동 제안 |

---

## 사용 방법 (3단계)

### 1️⃣ 대화 가져오기

**폰:** 카톡 캡처 → 위에서 아래 순서로 업로드 (최대 6장)  
**PC:** 대화 드래그 → 복사 → 붙여넣기  
**txt:** 카톡 → 대화 내용 저장 → 파일 업로드

### 2️⃣ 본인 선택 (필요할 때)

이름이 그대로 보이면 **「이 중에 너는?」** 버튼으로 본인만 골라 주세요.  
자동으로 「나」/「사용자」로 바뀝니다.

### 3️⃣ 🔥 호감도 확인하기

20~40초 정도면 AI 심층 리포트가 열려요.

## 안전한가요?

- 대화 **서버에 저장 안 함**
- 이름 **자동 익명화** 후 분석
- 결과만 화면에 표시
